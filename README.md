### Instalacja npm i redis-server na Linux

#### Zainstaluj npm:
```bash
sudo apt install npm
```

#### Zainstaluj redis-server:
```bash
sudo apt install redis-server
```

### dodaj executive do start.sh
```bash
chmod +x start.sh
```

### uruchom skrypt
```bash
./start.sh
```
